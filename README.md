# Coins Hockey
Coins Hockey или CH это
+ Рабочая физика
+ 26 видов монет
+ Реалистичные данные
+ Система кача
+ И просто способ забавно провести время

Прочитайть сохранение - Ctrl + R (read)
Сделать сохранение - Ctrl + S (save)
Начать,остановить запись - R (replay)
